# Handwritten Digit Recognition using CNN (GoogLeNet Inspired)

This project implements a Convolutional Neural Network using TensorFlow
to recognize handwritten digits from the MNIST dataset.

## Technologies Used
Python
TensorFlow
NumPy
Matplotlib

## Dataset
MNIST dataset from TensorFlow datasets.

## Model Architecture
The model uses a simplified GoogLeNet style architecture with
custom Inception modules.

## Features
- Image preprocessing
- CNN with Inception blocks
- Training and evaluation
- Visualization of test images

## Run the Project

Install dependencies

pip install -r requirements.txt

Run the program

python digit_recognition.py
